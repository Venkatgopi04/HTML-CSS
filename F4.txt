Working with table tag:
===============
HTML & HTML5 supports table tag.table tags are paired tag.The main 
objective of table tag is represent the data/information in tabular_format
or row and column format.

<table>
<tr>
<th></th>
</tr>
<tr>
<td></td>
</tr>
</table>

Working on List Tag:
=============
HTML & HTML5 supports List tags.List tags are paired tag which means combination of
an opening and closing tag.The main objective of List tag is reprent the content in list form


->orderlist ----><ol><li></li></ol>
->unorderlist ----><ul><li></li></ul>
